# 📱 Android Registration Form App

A modern **Android Registration Form Application** built using **Android Studio** implementing form validation, UI components, and user interaction features.

This project was created as a **Week 5 assignment** to demonstrate the use of various Android UI components such as TextInputLayout, RadioGroup, Checkbox, Spinner, AlertDialog, and Gesture Interaction.

# ✨ Features

## 1. Complete Registration Form
A full registration form built using **Material Design TextInputLayout**.

Input fields:
- Name
- Email
- Password
- Confirm Password

Features:
- Input validation
- Error messages
- Material Design UI

---

## 2. Advanced Validation

The application performs several input validations:

- Required field validation
- Email format validation
- Password confirmation check
- Real-time validation

Example validations:
- Email must follow valid email format
- Password and Confirm Password must match
- Fields cannot be empty

---

## 3. Selection Controls

### RadioGroup

Used to select **Gender**.

Options example:
- Male
- Female

---

### Checkbox

Used for selecting **Hobbies**.

Example hobbies:
- Coding
- Gaming
- Music
- Reading
- Traveling

Validation rule:
- User must select **minimum 3 hobbies**

---

## 4. Spinner & AlertDialog

### Spinner

Used to display a dropdown list for selecting data.

Example data:
- City
- Country
- Department

---

### AlertDialog

When the **Submit button** is clicked, a confirmation dialog appears.

Example message:

Options:
- Yes → Submit form
- No → Return to form

---

## 5. Gesture Interaction

This application implements **Long Press Gesture** on a button.

Example behavior:

This demonstrates gesture-based interaction in Android applications.
